# Application Set Map

- Candidate vision: company moment, operating tensions, candidate thesis, Mission Window Synchronizer, evidence transfer, scorecard, entry plan, executive questions, and application navigation.
- Resume: verified experience and capabilities selected for product portfolio leadership, mission systems, agentic AI, support-heavy products, and hands-on execution.
- Cover letter: role/company problem, four evidence threads, Mission Window Contract, complete candidate-vision URL, and first-120-day discussion invitation.
- Interview thesis brief: actual work, company moment, tensions, decision contract, decision rights, scorecard, proof map, entry sequence, and executive questions.
- 120-day entry plan: listening, decision architecture, live reviews, coaching, AI-assisted product tradecraft, learning return, and success evidence.
- Mission Window Review: practical two-page decision record for accelerate, scale, integrate, productize, prove, hold, or stop.

Each artifact reinforces the same thesis without repeating the same prose.
